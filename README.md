# 🏥 Debabhumi Medical Store

A modern and responsive medical store website built using HTML, CSS, and JavaScript.  
Designed to provide an online presence for a pharmacy with a clean healthcare-focused UI.

---

## 🚀 Live Demo

https://abhisekhmisra19-del.github.io/debabhumi-medical-store/

## ✨ Features
- Responsive design (Mobile + Desktop)
- Clean medical-themed UI
- Contact information section
- Smooth scrolling & modern layout
- Easy deployment using GitHub & Netlify
